  2. 코드 에디터를 열어_config.yml라는 파일을 열어본다.

  3. 76번줄에 theme : landscpae를 theme : icarus로 변경한다.

  4. 시연 구동이 되지를 않는다. 터미널에서 icarus 전용 _config.yml을 만들었다. 고로 구연시동이 되질 않음

  5. 크롬 웹을 통해, https://localhost:4000에 접속하면 테마가 바뀐 블로그를 확인 할 수 있다.
