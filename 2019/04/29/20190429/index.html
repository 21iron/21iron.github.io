  • Operators

    • Logical Operators(논리연산자)

      !NaN : Logical Not

    • Logical Or(||연산자)

      undefined || true :

      1. Or연산자는 왼쪽부터 truhy값을 찾아낸다
      2. truthy값을 찾은 경우, 해당값을 반환한다.
      3. 마지막에 도달할 경우, 마지막 값을 돌려준다.
    • Logical And(&&연산자)

      true && "" && 300 :

      1. 왼쪽부터 Falsy 값을 찾아나간다.
      2. Falsy 값을 찾은 경우, 해당값을 반환한다.
      3. 마지막에 도달할 경우, 마지막 값을 반환한다.
    • Ternary Operator(삼항 조건 연산자)

      true ? 1:2 :

      1. ? 앞 구문이 참, 1 변환
      2. ? 앞 구문이 거짓, 2 변환
