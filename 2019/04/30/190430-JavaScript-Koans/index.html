190430 JavaScript Koans

[바코004]
  • Array(10) : 10개의 배열을 만든다.
  • .push(a) : 작동 : 배열의 끝에 a를 추가 | 반환 : 추가 후 배열의 길이
  • .pop() : 작동 : 배열의 마지막 요소를 제거 | 반환 : 제거된 요소 반환
  • .unshift(a,b) : 작동 : 배열의 맨 앞쪽으로 새로운 요소 추가 | 반환 : 추가 후 배열의 길이 반환
  • .shift : 작동 : 배열의 첫번째 요소 제거 | 반환 : 제거된 요소 반환
  • .slice(a,b) : 작동 : 배열의 a부터 b까지 복사하여 새로운 배열 객체 생성 | 반환 : 복사한 새로운 배열 반환
  • .join() : 작동 : 배열의 모든 요소를 연결하여 하나의 문자열로 생성 | 반환 : 생성한 문자열 반환 ※ .join(-)으로 하면 요소를 연결하는 지점마다 -를 추가하게 된다.
